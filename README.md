## 🚀 Overview

> 🎙️ **Audio Transcriber & Translator** is a powerful cloud-based web app that allows users to:
>
> ✅ Upload and preview `.mp3` audio files  
> ✅ Transcribe speech to text using **AWS Lambda + S3**  
> ✅ Translate the text into **English 🇬🇧**, **Hindi 🇮🇳**, or **Telugu 🇮🇳** in real-time  
> ✅ Experience a modern, responsive, animated UI with smooth interactions  

All powered by **serverless cloud technology** for fast, scalable performance.

## ✨ Key Features

| 🔑 Feature                         | 💡 Description                                                                     |
|-------------------------------------|-----------------------------------------------------------------------------------|
| 🎙️ **Audio Upload & Playback**      | Upload `.mp3` files and control playback speed with an interactive audio player.  |
| 📝 **Speech to Text (Transcription)** | Uses **AWS Lambda & S3** to generate accurate transcriptions of spoken content.   |
| 🌐 **Translation**                   | Translates text into **English, Hindi, or Telugu** using **MyMemory Translation API**. |
| 🎨 **Sleek & Responsive UI**         | Engaging, mobile-friendly design with **AOS animations** for smooth interactions.  |
| ⏳ **Progress & Feedback**           | Includes animated loaders and proper error handling for better user experience.    |

## 🚦 How It Works — Step by Step

1. **📤 Upload**: Choose and preview an `.mp3` audio file.
2. **📝 Transcribe**: Click **Transcribe** to generate text via AWS backend.
3. **🌐 Translate**: Enter or use the transcribed text and select a target language.
4. **✅ Output**: The translated text is instantly displayed for copy or download.
